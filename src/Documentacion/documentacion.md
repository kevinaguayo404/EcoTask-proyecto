# 1. Descripción del Proyecto

## Objetivo

SimpleTask es una aplicación web creada con React que ayuda a organizar tareas de una manera sencilla. Su objetivo es que el usuario pueda llevar un control de sus actividades diarias sin complicaciones.

## Alcance

La aplicación está pensada para cualquier persona que quiera organizar mejor sus pendientes, ya sea para la escuela, el trabajo o actividades personales.

Toda la información se guarda en el navegador mediante LocalStorage, por lo que no es necesario crear una cuenta ni utilizar una base de datos.

## Características principales

- Agregar tareas.
- Marcar tareas como completadas.
- Eliminar tareas.
- Guardar automáticamente las tareas.
- Mostrar estadísticas.
- Mostrar una barra de progreso.
- Mostrar un consejo de productividad.

# 2. Manual de Usuario

## Requisitos

Para utilizar la aplicación solo se necesita:

- Un navegador web actualizado.
- Node.js instalado (si se desea ejecutar el proyecto desde Visual Studio Code).

## Instalación

1. Descargar o clonar el proyecto desde GitHub.

```bash
git clone https://github.com/kevinaguayo404/EcoTask-proyecto.git
```

2. Entrar a la carpeta del proyecto.

```bash
cd EcoTask-proyecto
```

3. Instalar las dependencias.

```bash
npm install
```

4. Ejecutar la aplicación.

```bash
npm run dev
```

5. Abrir el navegador en la dirección indicada por Vite.

## Uso de la aplicación

### Agregar una tarea

Escribe la tarea en el cuadro de texto y presiona el botón **Agregar**.

### Completar una tarea

Marca la casilla que aparece al lado de la tarea.

### Eliminar una tarea

Presiona el botón rojo de eliminar.

### Estadísticas

Las estadísticas muestran cuántas tareas hay en total, cuántas están terminadas y cuántas siguen pendientes.

### Barra de progreso

La barra aumenta automáticamente conforme se completan tareas.

# 3. Evaluación del Impacto Ambiental

SimpleTask es una aplicación pequeña que consume pocos recursos del equipo.

Como funciona directamente en el navegador y utiliza LocalStorage para guardar la información, no necesita un servidor o una base de datos para funcionar. Esto ayuda a reducir el consumo de recursos en comparación con aplicaciones más grandes.

Además, el proyecto está organizado por componentes, lo que facilita hacer cambios sin tener que modificar todo el programa. Esto ayuda a que el mantenimiento sea más sencillo y evita trabajo innecesario.

Aunque cualquier aplicación consume energía al ejecutarse, el impacto ambiental de SimpleTask es bajo debido a que es una aplicación ligera.

# 4. Informe de Mantenimiento

Durante el desarrollo se buscó que el proyecto fuera fácil de entender y modificar.

Para lograrlo se utilizaron componentes independientes, lo que permite cambiar una parte de la aplicación sin afectar las demás.

También se utilizó Git y GitHub para guardar el historial de cambios y tener un respaldo del proyecto.

Los nombres de las variables, funciones y componentes se eligieron para que fueran fáciles de entender y el código quedara más organizado.
